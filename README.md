# Sistem Inventaris Barang

Aplikasi berbasis Java Swing dan MySQL untuk mengelola data barang, transaksi, dan laporan.

## Fitur
- Login Admin & Petugas
- Data Barang
- Transaksi Peminjaman & Pengembalian
- Laporan
- Cek Stok Menipis (Subquery)
- Trigger Otomatis Stok

## Teknologi
- Java (PBO)
- MySQL / MariaDB
